# 现代计算器 - Python tkinter实现

## 概述
这是一个使用Python tkinter库创建的现代、美观的计算器应用程序。包含两个版本：
1. `calculator_final.py` - 完整功能的现代计算器
2. `simple_calculator.py` - 简化版本，确保基本功能

## 功能特点

### 基本功能
- 基本运算：加(+)、减(-)、乘(*)、除(/)、取模(%)
- 小数运算：支持小数点输入
- 清除功能：C（全部清除）、←（退格）
- 等于运算：= 按钮执行计算
- 连续运算：支持连续计算（如：1+2=3, +5=8）

### 高级功能
- 键盘支持：支持键盘输入数字和运算符
- 错误处理：除以零等错误处理
- 历史显示：显示当前计算表达式
- 现代界面：美观的配色方案和布局

### 界面设计
- 深色主题：深蓝色背景，高对比度显示
- 彩色按钮：不同功能的按钮使用不同颜色
- 响应式设计：按钮有悬停效果
- 清晰显示：大字体显示当前输入和结果

## 使用方法

### 运行计算器
```bash
# 运行完整版本
python3 calculator_final.py

# 运行简化版本
python3 simple_calculator.py
```

### 操作方式
1. **鼠标操作**：点击界面上的按钮
2. **键盘操作**：
   - 数字键：0-9
   - 运算符：+ - * / %
   - 小数点：.
   - 等于：Enter键
   - 清除：Esc键
   - 退格：Backspace键

### 计算示例
1. 基本计算：`5 + 3 = 8`
2. 小数计算：`3.14 * 2 = 6.28`
3. 连续计算：`10 / 2 = 5, + 3 = 8`
4. 错误处理：`5 / 0 = 错误: 除以零`

## 文件说明

### calculator_final.py
完整功能的现代计算器，包含：
- 历史显示区域
- 更完善的错误处理
- 更好的连续运算支持
- 更美观的界面设计

### simple_calculator.py
简化版本的计算器，包含：
- 所有基本功能
- 简化的界面
- 确保在大多数环境中的兼容性

## 技术要求
- Python 3.x
- tkinter库（通常包含在Python标准库中）
- 在Ubuntu/WSL中可能需要安装：`sudo apt-get install python3-tk`

## 代码结构
两个计算器都采用面向对象设计：
1. `__init__`：初始化界面和变量
2. `create_widgets`：创建界面组件
3. `bind_keyboard_events`：绑定键盘事件
4. `button_click`：处理按钮点击
5. `calculate`：执行计算逻辑
6. 其他辅助方法：更新显示、处理输入等
