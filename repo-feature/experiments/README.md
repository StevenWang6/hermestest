## How to Add New Experiments
1. Create a new directory under `experiments/`
2. Add a README.md describing the experiment
3. Implement the experiment code
4. Document results in `docs/results/`

## Experiment Guidelines
- Keep experiments focused and small
- Document assumptions and constraints
- Include verification steps
- Clean up after experiments are complete
