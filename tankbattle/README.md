# 🎮 Tank Battle Game

一个简单的坦克对战游戏，使用 HTML5 Canvas 和 JavaScript 开发。

## 🚀 功能特性

- 🎯 双人对战模式
- 🏗️ 可破坏的障碍物
- 💥 炮弹物理效果
- 🎨 像素艺术风格
- ⚙️ 可配置的游戏参数

## 📁 项目结构

```
tankbattle/
├── src/           # 源代码
│   ├── game.js    # 游戏主逻辑
│   ├── tank.js    # 坦克类
│   ├── bullet.js  # 炮弹类
│   └── map.js     # 地图类
├── assets/        # 资源文件
│   ├── images/    # 图片资源
│   └── sounds/    # 音效资源
├── docs/          # 文档
├── tests/         # 测试文件
├── index.html     # 主页面
├── style.css      # 样式文件
└── package.json   # 项目配置
```

## 🎮 游戏控制

### 玩家1 (红色坦克)
- **W/A/S/D**: 移动
- **空格键**: 发射炮弹

### 玩家2 (蓝色坦克)
- **方向键(↑↓←→)**: 移动
- **Enter键**: 发射炮弹

## 🛠️ 开发指南

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm start
```

### 构建项目
```bash
npm run build
```

### 运行测试
```bash
npm test
```
